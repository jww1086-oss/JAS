# 🧠 Anti-Gravity Safety System - Project Memory (v25.0)

이 문서는 **Anti-Gravity Safety System (KOSHA Smart Safety)** 프로젝트의 현재 상태, 핵심 아키텍처, 구현된 기능 및 유지보수 정보를 기록한 '프로젝트 메모리'입니다.

## 🚀 프로젝트 개요
- **이름**: Anti-Gravity Safety System (KOSHA 스마트 안전 점검 시스템)
- **버전**: v25.0 (Unified Card UI Edition)
- **목적**: 현장 근로자가 모바일기기를 통해 실시간으로 위험성 평가를 실시하고, 결과를 구글 시트에 즉시 동기화하며, 고품질 카드형 리포트를 즉시 생성하는 시스템.
- **철학**: "Consistency in Quality" - 미리보기부터 결과 조회까지 동일한 프리미엄 경험 제공.

## 🛠 기술 스택 (Tech Stack)
- **Frontend**: HTML5, Vanilla JavaScript, CSS3 (Premium Card Design System) - **Vercel 배포 (`jungbu.vercel.app`)**
- **Backend/API**: Google Apps Script (GAS) - **API 서비스 및 데이터 처리**
- **Database**: Google Sheets (실시간 동기화)
- **Library**: Lucide Icons, Signature Pad JS, html2pdf.js

## 🏗 시스템 아키텍처 (Hybrid Cloud)
1.  **Vercel Frontend (`jungbu.vercel.app`)**:
    - 사용자가 직접 접속하는 메인 UI 서버.
    - `index.html`, `app.js`, `style.css`로 구성된 독립적 프론트엔드.
2.  **GAS Backend API (`script.google.com`)**:
    - 구글 시트와 통신하여 데이터를 정제하고 JSONP 형식으로 응답.
    - 점검 결과 삽입(`doPost`) 및 마스터 데이터/로그 조회(`doGet`) 담당.
3.  **데이터 동기화**:
    - `fetchJSONP`를 통해 Vercel(Client)과 GAS(Server) 간의 CORS 제한 없이 실시간 통신.
