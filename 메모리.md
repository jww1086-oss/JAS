## 🛠 구글 시트 데이터 구조 (v25.6 표준)
정밀한 위해 위험요인 관리를 위해 총 **16개 컬럼 (A~P열)** 구조를 사용합니다.

| 열 | 헤더명 | 설명 |
| :--- | :--- | :--- |
| A | 일시 | 점검 일시 (자동 생성) |
| B | 부서명 | 점검 대상 부서 |
| C | 작업명 | 수행 중인 작업명 |
| D | 점검자 | 점검 실시자 성명 |
| E | 작업단계 | 리스크 관리 단계 |
| F | 위험요인 | 식별된 유해/위험 요인 |
| G | 현재안전조치 | 현재 이행 중인 안전 조치 |
| H | 개선대책 | 추가 발굴된 개선 대책 |
| I | 현재_빈도 | 조치 전 발생 가능성 (I열) |
| J | 현재_강도 | 조치 전 사고 중대성 (J열) |
| K | 현재_위험도 | 조치 전 최종 위험 수준 (K열) |
| L | 잔류_빈도 | 조치 후 발생 가능성 (L열) |
| M | 잔류_강도 | 조치 후 사고 중대성 (M열) |
| N | 잔류_위험도 | 조치 후 최종 위험 수준 (N열) |
| O | 사진URL | 현장 사진 데이터 (Base64) |
| P | 서명URL | 점검자 서명 데이터 (Base64) |
