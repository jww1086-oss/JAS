# 🧠 Anti-Gravity Safety System - Project Memory

이 문서는 **Anti-Gravity Safety System (KOSHA Smart Safety)** 프로젝트의 현재 상태, 핵심 아키텍처, 구현된 기능 및 유지보수 정보를 기록한 '프로젝트 메모리'입니다.

## 🚀 프로젝트 개요
- **이름**: Anti-Gravity Safety System (KOSHA 스마트 안전 점검 시스템)
- **목적**: 현장 근로자가 모바일기기를 통해 실시간으로 위험성 평가를 실시하고, 결과를 구글 시트에 즉시 동기화하며, 법적 준수 사항을 체크하는 가볍고 강력한 웹 앱.
- **철학**: "3-Click Workflow" - 최소한의 조작으로 최대의 안전 확보.

## 🛠 기술 스택 (Tech Stack)
- **Frontend**: HTML5, Vanilla JavaScript, CSS3 (Premium Design System)
- **UI/UX Assets**: Lucide Icons, Google Fonts (Inter, Outfit), Signature Pad JS
- **Backend/DB**: Google Apps Script (GAS) & Google Sheets
- **Deployment**: Vercel (GitHub 연동 CI/CD 추천)
- **Data Format**: JSONP (CORS 우회 및 실시간 데이터 통신용)

## 🏗 시스템 아키텍처
1.  **데이터 마스터 (Google Sheets)**:
    - `위험성_마스터`: 부서별/작업별 위험 요인 및 개선 대책 수록.
    - `실시_로그`: 점검 결과, 사진 URL, 디지털 서명 데이터 저장.
    - `사용자_명단`: 부서원 명단 및 직책 정보.
2.  **API 레이어 (GAS - Code.gs)**:
    - `doGet`: 데이터 조회 (JSONP 응답).
    - `doPost`: 점검 결과 업로드 및 이미지(Base64) 처리.
3.  **클라이언트 (app.js)**:
    - 실시간 데이터 동기화 및 캐싱.
    - 단계별 상태 관리 (Step 1 ~ Step 4).
    - 모바일 최적화 애니메이션 및 뒤로가기 제어 (`popstate`).

## ✨ 주요 구현 기능
- [x] **프리미엄 대시보드**: 9개의 주요 메뉴 그리드 및 다이내믹 배너.
