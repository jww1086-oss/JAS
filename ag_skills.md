# 🛠 Antigravity Skills for Power Plant Risk Assessment (v35.8.14)

이 문서는 본 프로젝트에 적용된 AI 에이전트의 고도화된 기술적 역량과 구현 기술을 기록합니다.

## 1. 지능형 데이터 엔지니어링 (Data Intelligence)
- **Intelligent Key Mapping**: 스펙에 명시되지 않은 컬럼명이라도 정규식과 의미론적 분석을 통해 `부서`, `작업`, `위험요인` 등을 자동 탐지하여 매핑.
- **NFC Unicode Normalization**: 한글 자모 분리 현상을 원천 차단하기 위해 모든 입출력 데이터에 유니코드 정규화 적용.
- **Auto Fill-Down Engine**: 병합된 셀이나 반복 생략된 상위 항목 데이터를 지능적으로 하위 레코드에 복제하여 데이터 무결성 확보.

## 2. 스마트 시인성 최적화 (UI/UX Intelligence)
- **Flat Layout Conversion**: 복잡한 Box-in-Box 구조를 리스트 기반의 Flat 구조로 재설계하여 모바일 가독성 극대화.
- **Korean Word Preservation**: `keep-all` 기반의 단어 단위 줄바꿈 로직을 적용하여 한 글자 고립 현상(Orphans) 해결.
- **Intelligent Deduplication**: 제목(위험요인)과 내용(안전조치) 간의 텍스트 중복을 실시간으로 분석하여 필터링하는 정보 다이어트 로직.

## 3. 안정성 및 성능 최적화 (System Stability)
- **Cache-Busting Deployment**: 서비스 업데이트 시 전역 버전 쿼리를 강제 갱신하여 클라이언트 캐시 문제를 해결하는 안정적 배포 스킬.
- **Gap-Analysis Automation**: 미이행 조치 사항을 개선 대책 섹션으로 자동 전이하고 시각적으로 강조하는 워크플로우 제어 기술.

**※ 본 기술 역량은 '발전소 위험성평가' 시스템의 안정성과 프리미엄 품질을 유지하는 핵심 동력입니다.**
